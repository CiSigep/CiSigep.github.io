About Me

I'm a developer who has been trying to get myself back into the industry for a while now. I enjoy Video Games, Tabletop Games, and new gadgets. I got into development because I spent most of my time on the computer and wanted to know how it worked software-wise. Development is very interesting to me as it increases my understanding of computers and how they can make pretty much anything easier for those who use them. Nowadays, web applications are getting more and more prominent as they don't need to be installed and you can access them by web browser. I'd like to learn as much as I can about them.

I am familiar with Object-Oriented Languages such as C# and Java, for which I am an Oracle Certified Associate (OCA) for Java 8 SE. Several of the technologies I am familiar with and have worked on projects for are all Java EE technologies like WebLogic Server, Java Server Pages (JSPs), Spring, Hibernate, and Maven. Picking up new technologies is something I am very good at. Show me something I don't know and how to work it, I can figure it out and make use of it rather quickly.
